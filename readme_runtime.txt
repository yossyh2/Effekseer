■　概要
Effekseerをダウンロードしていただきありがとうございます。
本パッケージはDirectXやOpenGLを使用しているアプリケーションで、エフェクトを再生できるようにするランタイムが保存されています。

デザイナー向けのエフェクトを作成するためのツールやエフェクトのサンプルは保存されていませんので、
それらが必要な方はツールのパッケージをダウンロードするようお願いします。


■　ディレクトリ構成
-Compiled
コンパイル済みのlibファイルが保存されています。
VisualStudio2012、2013のみ対応しています。
ただし、OpenGL版は2012には対応していません。

コンパイルオプションは
デバッグは、マルチスレッドデバッグ
リリースは、マルチスレッド

になっています。

-src
ランタイムのソースコードが保存されています。

-RuntimeSample
ランタイムを組み込んだサンプルが保存されています。

-Help
組込み方法について記述されているヘルプが保存されています。
